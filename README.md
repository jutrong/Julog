## Project 소개

1. 9. 11 ~ 진행중

- 개인 이력서 겸 블로그, 나의 작업물 담는 공간을 gatsby.js로 제작하였습니다.
- 자바스크립트 애니메이션 라이브러리 Gsap를 사용하여 동적인 애니메이션 메뉴바를 구현하였습니다.
- 마크다운 블로그 포스팅
- Desktop, tablet, mobile 사이즈에 맞게 반응형으로 제작하였습니다.
- Introduce | Project | Skills | Experience 순서로 이력을 기재했습니다.
